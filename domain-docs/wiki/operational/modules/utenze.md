# Modulo Utenze

## Scopo

Il modulo Utenze consente consultazione anagrafiche utenza, import, routing visure e anomalie collegate.

## Cosa puo fare l'operatore

- leggere dati di un'utenza e campi rilevanti
- verificare esiti import e scarti
- capire anomalie di routing visure
- trovare informazioni collegate a particelle o soggetti

## Dati o input tipici

- codice utenza o identificativo
- periodo import
- codice fiscale o partita IVA
- riferimento visura o pratica collegata

## Pagine principali

- `/utenze/import`: esiti import
- `/utenze/visure-routing-anomalies`: anomalie routing visure

## Prossimi passi

Indica codice utenza, anomalia o pagina di partenza e ti guido nella consultazione.
