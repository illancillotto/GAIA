# Modulo Inaz

## Scopo

Il modulo Inaz supporta consultazione collaboratori, giornate lavorative, organigramma e dati HR collegati.

## Cosa puo fare l'operatore

- leggere scheda collaboratore e responsabilita
- orientarsi nell'organigramma Inaz
- interpretare dati giornalieri e anomalie
- trovare responsabili e operatori collegati

## Dati o input tipici

- nome collaboratore o ID
- data o periodo
- nodo organigramma o responsabile

## Pagine principali

- `/inaz/collaboratori`: schede collaboratore
- `/inaz/organigramma`: albero organizzativo Inaz

## Prossimi passi

Indica collaboratore, data o area organigramma e ti guido nella consultazione.
