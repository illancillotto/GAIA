# Modulo Riordino

## Scopo

Il modulo Riordino gestisce pratiche di riordino, fasi workflow, documenti e configurazioni di processo.

## Cosa puo fare l'operatore

- leggere stato e fase di una pratica
- capire documenti richiesti e timeline
- interpretare configurazioni che influenzano il workflow
- orientarsi tra pratiche e impostazioni

## Dati o input tipici

- ID pratica riordino
- stato o fase corrente
- tipo pratica o configurazione

## Pagine principali

- `/riordino/pratiche`: elenco e dettaglio pratiche
- `/riordino/configurazione`: regole e impostazioni workflow

## Prossimi passi

Indica ID pratica o fase che ti interessa e ti guido nella consultazione.
