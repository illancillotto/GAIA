# Workflow: catasto owner search

## Obiettivo

Aiutare l'operatore a trovare il proprietario o intestatario di un terreno nel modulo Catasto.

## Ordine di lavoro consigliato

1. capire se la richiesta e documentale o operativa
2. verificare se l'utente ha almeno uno tra:
   - comune + foglio + particella
   - nominativo
   - codice fiscale
   - partita IVA
3. se i dati mancano, chiedere solo gli slot mancanti
4. se i dati ci sono, guidare verso la capability o il tool corretto

## Cosa non fare

- non dire che manca documentazione se il problema reale e che mancano identificativi
- non inventare risultati o nominativi
- non trasformare una richiesta operativa in una risposta generica di supporto

## Risposta attesa in caso di dati mancanti

Ciao. Per aiutarti a trovare il proprietario di un terreno mi servono almeno comune, foglio e particella, oppure un nominativo, codice fiscale o partita IVA. Se me li indichi, posso guidarti nella ricerca Catasto in modo operativo.
