# Docs Lookup

## Scopo

Usare il corpus operativo del Wiki come prima fonte quando una richiesta e documentale, operativa o di orientamento, ma non richiede ancora un dato live specifico.

## Regola

- preferire pagine operative del modulo o della capability corrente
- usare retrieval generale solo se il documento operativo dedicato non basta
- non degradare subito in `docs_not_found` quando il task e chiaramente riconoscibile
